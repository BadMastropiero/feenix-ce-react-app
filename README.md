# FEENIX CE Client

This repo was bootstraped with: [Vite](https://github.com/vitejs/vite), [React](https://reactjs.org/), [Typescript](https://www.typescriptlang.org/), [Eslint](https://eslint.org/), [Prettier](https://prettier.io/), [Husky](https://typicode.github.io/husky/), [Lint Staged](https://github.com/okonet/lint-staged), [Styled Components](https://styled-components.com/), [React Query](https://tanstack.com/query), [Zod](https://zod.dev/), [Docker](https://www.docker.com/).

## Quick start

Clone the repo and run:

```bash
yarn install
```

```bash
yarn dev
```
